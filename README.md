# Gemini Hub

A simple, single-file HTML5 chat interface for Google's Gemini 2.5 Flash API with voice recording support.

## ✨ Features
- **Single File:** Pure HTML, CSS, and JS. No installation or setup required.
- **Voice Input:** Record and send voice notes directly from your browser.
- **Web Search:** Automatically searches the web for real-time information.

## 🚀 How to Use
1. Download or copy the code into a file named `index.html`.
2. Open the file in a text editor, find `const API_KEY = "your-key";`, and replace `"your-key"` with your real Google AI Studio key.
3. Double-click `index.html` to open it in any browser and start chatting!

## ⚠️ Security Warning
Do not upload this file to a public GitHub repository or public website with your real API key saved inside it, or anyone will be able to view and use your key.
